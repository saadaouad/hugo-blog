Create React App

Oct 28, 2017 00:00 · 268 words · 2 minutes read

You don’t need to install or configure tools like Webpack or Babel… They are preconfigured and hidden so that you can focus just on the code!

Getting Started

Installation

npm install -g create-react-app

or

yarn global add create-react-app

Create app

To create a new app make you sure to run

create-react-app my-app
cd my-app

Try to open the current folder in your editor, you should see the initial structure below.

Run app in development environment

npm start

or

yarn start

The app should be compiled successfully

Compiled successfully!

You can now view my-app in the browser.

  Local:            http://localhost:3000/
  On Your Network:  http://192.168.1.150:3000/

Note that the development build is not optimized.
To create a production build, use yarn build.

The page will automatically reload if you make changes to the code.

Builds app for production

Builds the app for production to the build folder, using

npm run build

or

yarn build
