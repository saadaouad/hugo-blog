Know your personal information dashboard for your terminal

Jun 10, 2018 00:00 · 245 words · 2 minutes read

WTFutil is a personal terminal based dashboard utility, designed for displaying important and daily data.

Quick start

This tool is only compatible with Go versions 1.9.2 or later. It currently does not compile with gccgo.

➜  ~ go get -u github.com/senorprogrammer/wtf

➜  ~ cd go/src/github.com/senorprogrammer/wtf

➜  wtf make install

➜  wtf git:(master) make run
go build -o bin/wtf
bin/wtf
